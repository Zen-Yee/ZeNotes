# Git

## Git Cheatsheet

|Command|Description|
|----|----|
|git status|Show modified files in working directory, staged for next commit|
|git add [file]|Add a file to staging|
|git add .|Add all modified file to staging|
|git commit -m "[commit message]"|Add file in staging to local repo|
|git reset [file]|Remove file from staging|
|git log|View commits history. Press q to quit. |
|git pull|Pull changes from remote/online repo to local repo|
|git push|Push changes from local repo to remote/online repo|
