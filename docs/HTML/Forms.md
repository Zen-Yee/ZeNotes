# Web Forms

## How to use Forms in HTML?
```
<form action="(target endpoint)" method="(HTTP method: GET/POST)">
  <label for="UserName">Username:</label>
  <input type="text" id="UserName" name="UserName" value="John">
  <input type="submit" value="Submit">
</form> 
```
