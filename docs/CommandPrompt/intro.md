# Command Prompt

## Files & Folder Management
|Command|Description|
|----|----|
|cd|Change directory.|
|cd \ |Return to root. (e.g. C:\ ) |
|cd ..|Moves up one directory level from your current location. |
|*drive_letter*:|Switches to a different drive.|
|dir|Displays files and folders in the current directory|
|mkdir|Create folder.|

## Disks Management
|Command|Description|
|----|----|
|fsutil volume diskfree *drive_letter*:|Displays the total free space, total bytes, and total available free bytes for the specified drive.|

## Network Management
|Command|Description|
|----|----|
|ipconfig|Displays current TCP/IP network configuration values, including IP address, subnet mask, and default gateway|
|ping|Sends ICMP requests to the target host to check its availability|
